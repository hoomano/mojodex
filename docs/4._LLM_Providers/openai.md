# Open AI

Setup the API key in your `.env` file:

```bash
OPENAI_API_KEY=your-api-key
```

!!! note "Switching from OpenAI to Azure OpenAI Services"
    This is made easy by changing the `MAIN_MODEL_API_TYPE` in your `.env` file.
